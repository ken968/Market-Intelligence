XAUUSD tracker 

aku lagi ada proyek AI Dashboard untuk prediksi harga Emas (XAUUSD). Saya ingin kamu memahami struktur dan logika seluruh codebase saya agar kamu bisa membantu saya dalam debugging, optimasi, atau penambahan fitur ke depannya.

Deskripsi Proyek: XAUUSD CORE TERMINAL
Ini adalah aplikasi dashboard berbasis Streamlit yang berfungsi untuk memantau harga emas secara real-time, menganalisis sentimen berita global menggunakan NLP, dan melakukan prediksi harga ke depan (Multi-Range Forecast) menggunakan model Deep Learning LSTM. Aplikasi ini didesain dengan estetika 'Financial Terminal' yang premium (Deep Navy & Brass theme).

Struktur Folder & File:
XAUUSD/
├── .venv/                  # Virtual Environment Python
├── app.py                  # File utama UI Dashboard (Streamlit)
├── data_fetcher.py         # Mengambil data harga & makro dari Yahoo Finance
├── sentiment_fetcher.py    # Mengambil berita dari NewsAPI & analisis sentimen NLP
├── train_ultimate.py       # Script untuk training model LSTM
├── gold_macro_data.csv     # Database mentah harga & indikator makro
├── gold_global_insights.csv # Database final (gabungan makro + sentimen)
├── latest_news.json        # Data berita terbaru untuk ditampilkan di UI
├── scaler.pkl              # Objek normalisasi data (MinMaxScaler)
└── gold_ultimate_model.h5  # Model AI yang sudah dilatih (Keras/TensorFlow)

Detail Logika Tiap Komponen:
1. data_fetcher.py: Mendownload data historis 10 tahun untuk Gold (GC=F), DXY (Indeks Dollar), VIX (Volatilitas), dan Yield 10Y US Treasury via yfinance.
2.sentiment_fetcher.py: Mengambil berita finansial hanya dari sumber terpercaya (Bloomberg, Reuters, WSJ, dll) melalui NewsAPI. Menggunakan TextBlob untuk menentukan skor polaritas sentimen (-1 s/d 1) yang kemudian digabung ke database harga.
3. train_ultimate.py:
Menggunakan arsitektur LSTM 3-layer (100-50-25 units) dengan Dropout untuk mencegah overfitting.
Menggunakan 60 hari data historis untuk memprediksi harga hari berikutnya.
Menyimpan scaler.pkl agar proses inference di dashboard konsisten dengan hasil training.
4. app.py:
Dashboard UI: Menggunakan custom CSS untuk menyembunyikan elemen bawaan Streamlit yang kaku dan menggantinya dengan kartu industrial.
Sync Logic: Memiliki tombol untuk menjalankan fetcher produk secara otomatis.
Training UI: Memiliki fitur untuk melakukan re-train model langsung dari dashboard dengan log real-time yang bersih dari kode ANSI.
Recursive Forecasting: Menggunakan logika prediksi rekursif untuk menghasilkan ramalan harga dari 1 hari hingga 1 tahun ke depan.

Teknologi yang Digunakan:
Frontend: Streamlit, Plotly, HTML/CSS.
Backend/Processing: Pandas, NumPy.
Machine Learning: TensorFlow/Keras (LSTM), Scikit-Learn (MinMaxScaler), TextBlob (NLP).
Data Source: Yahoo Finance API & NewsAPI.
